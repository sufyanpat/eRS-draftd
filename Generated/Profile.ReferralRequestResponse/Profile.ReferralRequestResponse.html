    The following structure definitions are available in this profile.

    NameTypeDescription & Constraints
    eRS-ReferralRequest-1ReferralRequestThis ReferralRequest resource has been constrained to support the NHS e-Referral Service requirements.
    eRS-Patient-1PatientThis Patient resource carries the patient demographic details, as a supporting resource for the NHS e-Referral Service project.
    eRS-DocumentReference-1DocumentReferenceThis DocumentReference resource has been constrained to support the NHS e-Referral Service requirements.
    eRS-Appointment-1AppointmentThis Appointment resource has been constrained to support the NHS e-Referral Service requirements.
    eRS-Practitioner-1PractitionerThis Practitioner resource has been constrained to support the NHS e-Referral Service requirements.
    eRS-HealthcareService-1HealthcareServiceThis HealthcareService resource has been constrained to support the NHS e-Referral Service requirements.
    eRS-Location-1LocationThis Location resource has been constrained to support the NHS e-Referral Service requirements.
    eRS-OperationOutcome-1OperationOutcomeThis OperationOutcome resource supports a collection of error, warning or information messages that result from an NHS e-Referral Service request.
